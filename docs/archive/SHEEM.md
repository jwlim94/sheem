# Sheem (쉼)

> 위치 기반 ASMR 사운드를 경험하는 멀티플레이어 힐링 웹 앱

---

## 컨셉

사용자가 3D 게임 월드에서 캐릭터를 조종하며 돌아다니고, **위치와 환경에 따라 다른 ASMR 사운드**를 경험하는 힐링 웹 앱.

### 핵심 아이디어
- 현실 세계처럼 위치에 따라 소리가 다르게 들림
  - 예: 비 오는 날 → 처마 밑 빗소리, 아스팔트 위 빗소리, 차 안 빗소리 등
- 같은 소리를 좋아하는 사람들이 자연스럽게 모임
- 모여서 대화하며 각자 할 일 하는 힐링 공간

### 차별점
- 기존 ASMR 앱 (Noisli, Calm): 버튼 눌러서 소리 재생
- **Sheem**: 공간적 경험(spatial experience) + 소셜 요소

## 사운드 시스템 (확정)

### 핵심 원리: Spatial Audio (공간 오디오)

현실 세계의 소리 시스템을 반영한 설계:

1. **거리 기반 볼륨 (Distance-based Volume)**
   - 소리 소스에 가까울수록 볼륨 증가
   - 멀어질수록 자연스럽게 감소
   - 갑작스러운 전환 없음

2. **레이어링 (Layering)**
   - 여러 소리가 동시에 들림
   - 위치에 따라 각 소리의 믹스 비율이 달라짐
   - 예: 캠프파이어 근처 → 장작 소리 80% + 빗소리 20%

3. **소리 소스 (Sound Sources)**
   - 각 소리 소스는 3D 공간에 위치함
   - 영향 범위(radius)를 가짐
   - 범위 내에서 거리에 따른 그라데이션 볼륨

```
예시: 플레이어가 캠프파이어에 다가가는 경우

[멀리] 빗소리 100% | 장작소리 0%
   ↓
[중간] 빗소리 60%  | 장작소리 40%
   ↓
[가까이] 빗소리 20% | 장작소리 80%
```

### 기술: Web Audio API - Spatial Audio (3D Positional Audio)

## 월드 구조 (확정)

메이플스토리 스타일의 월드 구조:

```
🗺️ 월드맵
├── 🌧️ 비 월드 (MVP)
│   ├── 처마 밑 거리
│   ├── 아스팔트 도로
│   ├── 창가 카페
│   └── ...
├── 🔥 캠프 월드 (추후)
└── 🎵 클래식 월드 (추후)
```

### 이동 방식
- 걸어서 연결된 맵으로 이동
- 월드맵 열어서 클릭 → 텔레포트
- 포털 통해서 이동
